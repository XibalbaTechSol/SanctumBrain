## 🧠 Architectural Decisions
1. **Sovereign VPS First**: All heavy compute and intelligence resides on a user-owned VPS (Sanctum Guard).
2. **Aura Neural Core**: The primary agentic layer (rebranded from Hermes) that orchestrates PARA memory, VPS infrastructure, and Kinetic UI.
3. **Gemma 4 Edge Inference**: A specialized inference layer in the client handoff node that "paints" high-fidelity UI manifests from abstract intentions.
4. **Kinetic UI Synthesis**: Interfaces are generated dynamically via AGUI Manifests to collapse the "UI Wavefunction."
5. **Adaptive UI Registry**: Persistent storage for reusable, agent-modified UI components (Banking, Weather, etc.) for high-fidelity adaptive presentation.
6. **Hybrid Agentic Loop**: Combines local SLM (PII scrubbing) with LangGraph (Frontier reasoning).
7. **Unified Launcher**: Robust root-level orchestration (`start_sanctum.sh`) with SQLite fallbacks.
8. **Sovereign SaaS Model**: Tiered monetization with enforced feature gating for specialized neural tools.
